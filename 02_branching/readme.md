# Git Branching

Branching allows you to work on different features without affecting the main code.

---

## What is a Branch?

A branch is a separate version of your project where you can make changes safely.

---

## List Branches

```bash
git branch
```

---

## Create Branch

```bash
git branch feature-name
```

---

## Switch Branch

```bash
git switch feature-name
```

---

## Create + Switch

```bash
git switch -c new-feature
```

---

## Delete Branch

```bash
git branch -d branch-name
```

---

## Rename Branch

```bash
git branch -m old-name new-name
```

---

## HEAD

HEAD points to the current branch you are working on.

---

## Summary

* Branch = parallel development
* Helps avoid breaking main code
* Important for team work

---

💡 Always create a new branch for new features.
